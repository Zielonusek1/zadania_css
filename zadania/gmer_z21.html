ID - Użycie identyfikatora

Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID to skrót od identyfikatora, który jest unikatowym identyfikatorem dla elementu HTML.
Jaki jest cel stosowania ID? ID służy do identyfikacji pojedynczego elementu HTML na stronie, aby można było na niego wskazać bezpośrednio.
Jakie jest znaczenie znaku # Znak # oznacza, że następuje identyfikator.
Jak następuje wywołanie ID? ID jest wywoływane poprzez umieszczenie znaku # przed nazwą identyfikatora w wywołaniu selektora.
Uwagi na temat stosowania ID
  • Różnica między id a class - identyfikator może wystąpić tylko raz w całym dokumencie, natomiast klasę można powielać do woli.
  • Elementy oznaczone ID muszą być unikatowe, tzn., że na jednej stronie może wystąpić tylko 1 element z takim ID.
  • Nie należy nadużywać identyfikatorów, ponieważ mogą one być przydatne w odwoływaniu się do nich w skryptach.
